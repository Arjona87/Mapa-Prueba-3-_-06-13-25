Instrucciones para el archivo XLSX:

  • El archivo debe contener columnas con nombres como: Latitud, Longitud, Nombre, Descripción, Categoría
  • Las coordenadas deben estar en formato decimal (ej: 20.6597, -103.3496)
  • Las categorías pueden ser: seguridad, salud, educacion, transporte, turismo, general
  • Descarga el archivo de ejemplo para ver el formato correcto
